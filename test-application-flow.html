1Start Your Application

Mở terminal và chạy:

npm start

App sẽ chạy tại: http://localhost:3001

2Login as Candidate

Đăng nhập với tài khoản ứng viên (candidate role)

💡 Nếu chưa có tài khoản, đăng ký mới với role = candidate

3Upload CV

1. Vào trang Hồ sơ / Profile

2. Tìm phần CV/Hồ sơ

3. Click Tải CV lên và chọn file PDF

4. Đợi upload thành công

✅ CV sẽ được lưu vào S3 bucket: kfc-cv-bucket

4Apply for a Job

1. Vào trang Công việc / Job Listing

2. Chọn một công việc bất kỳ

3. Click Gửi CV ngay

4. Xác nhận trong popup

✅ Application sẽ được lưu vào DynamoDB table: Applications

5Check Browser Console

Mở Developer Tools (F12) và xem Console:

✅ Application submitted: { applicationId: "APP-20260313-XXXXXXXX", jobId: "...", candidateId: "...", cvUrl: "https://kfc-cv-bucket.s3.ap-southeast-1.amazonaws.com/..." }

6Login as Employer

1. Logout khỏi tài khoản candidate

2. Login với tài khoản employer (người đăng job đó)

3. Vào trang Hồ sơ ứng tuyển / Applications

4. Chọn job vừa apply

✅ Bạn sẽ thấy CV của candidate xuất hiện trong danh sách

7View CV

1. Click vào application để xem chi tiết

2. Click Xem CV hoặc Tải xuống

✅ CV sẽ hiển thị trong modal hoặc download về máy

⚠️ Troubleshooting

Nếu gặp lỗi "Failed to submit application":

  • Kiểm tra đã upload CV chưa
  • Mở Console (F12) xem error message chi tiết
  • Kiểm tra JWT token còn hạn không

Nếu không thấy application:

  • Đảm bảo đăng nhập đúng employer account (người đăng job)
  • Refresh trang Applications
  • Kiểm tra DynamoDB table Applications
